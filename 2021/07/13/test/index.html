1. 前言

在开始教程之前，你可以了解一下使用gitee pages服务搭建博客有哪些好处：

  • gitee 是国内的网站，相较于 github 访问速度更快；
  • 全是静态文件，访问速度快；
  • 免费方便，不用花一分钱就可以搭建一个自由的个人博客，不需要服务器不需要后台；
  • 数据绝对安全，基于gitee的版本管理，想恢复到哪个历史版本都行；
  • 博客内容可以轻松打包、转移、发布到其它平台；
  • 等等。
